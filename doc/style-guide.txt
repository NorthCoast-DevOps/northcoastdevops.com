Color Palette

	1.	Light Gray: #f0f0f0
	•	Usage: Primary background color.
	•	Description: Clean and neutral, providing a fresh canvas for content.
	2.	Dark Gray: #424245
	•	Usage: Primary text color and content.
	•	Description: Softer than pure black, offering good readability.
	3.	Almost Black: #1d1d1f
	•	Usage: Navigation background (with 0.8 opacity) and header text.
	•	Description: Strong contrast for navigation and headings.
	4.	Ocean Blue: #3498db
	•	Usage: Primary interactive elements.
	•	Description: Vibrant and engaging color for buttons and interactive elements.
	5.	Medium Blue: #2980b9
	•	Usage: CTA buttons and hover states.
	•	Description: Slightly darker shade for visual feedback.
	6.	Secondary Light Gray: #ecf0f1
	•	Usage: Feature callouts and secondary sections.
	•	Description: Subtle contrast from primary background.
	7.	Bright Blue: #00a8ff
	•	Usage: Navigation hover states and links.
	•	Description: Bright, attention-grabbing accent color.

Style Guide

Backgrounds

	•	Primary Background: #f0f0f0 (Light Gray) – Clean and neutral base for content.
	•	Secondary Background: #ecf0f1 (Secondary Light Gray) – Subtle contrast for feature sections or callouts.

Text Colors

	•	Primary Text: #424245 (Dark Gray) – Main body text color, offering good readability.
	•	Header Text: #1d1d1f (Almost Black) – For headings, providing clear visual hierarchy.
	•	Link Text: #00a8ff (Bright Blue) – Distinctive color for hyperlinks.

Buttons

	•	Primary Button: #3498db (Ocean Blue) – Used for main call-to-action buttons.
	•	Button Hover: #2980b9 (Medium Blue) – Color change on hover for interactive feedback.
	•	Button Text: White (#FFFFFF) – Ensures readability over button colors.

Borders and Dividers

	•	Light Border: #E0E0E0 – Subtle dividers or borders in sections to add structure without being visually heavy.
	•	Dark Border: #2A2E35 (Dark Navy) – Used sparingly, such as around images or key content blocks.

Icons and Links

	•	Icon Color: #1DA1DB (Ocean Blue) – Consistent with button color, for a unified look.
	•	Link Color: #FF914D (Sunrise Orange) – Stand-out color for links, matching the warm, inviting tones of your brand.

Animations

1. Landing Page Content
   • Fade In: 1.5s ease-in
   • Transform: translateY(20px) to 0

2. Scroll Arrows
   • Down Arrow: fadeInAndBounce animation (3s infinite)
   • Up Arrow: pulse animation (2s infinite)

UI Components

1. Cards
   • Background: White
   • Border: 1px solid var(--light-gray)
   • Border Radius: 8px
   • Box Shadow: 0 4px 6px rgba(0, 0, 0, 0.1)
   • Hover: translateY(-5px) with increased shadow

2. Buttons
   • Background: var(--ocean-blue)
   • Hover: var(--medium-blue)
   • Text: White
   • Padding: 12px 24px
   • Border Radius: 6px

3. Navigation
   • Background: rgba(29, 29, 31, 0.8)
   • Text: Linear gradient from white to light gray
   • Hover: var(--bright-blue)

4. Feature Callouts
   • Background: var(--secondary-light-gray)
   • Border Radius: 8px
   • Box Shadow: 0 4px 6px rgba(0, 0, 0, 0.1)
   • Padding: 30px

Responsive Design
   • Mobile Breakpoint: 768px
   • Height Breakpoint: 400px (scroll arrows)
   • Mobile Typography Scale:
     - h1: 2.5rem
     - h2: 2rem
     - h3: 1.3rem
     - Body: 1rem
     - Lead: 1.1rem

Additional Notes

	•	Button Hover Effects: Lighten the primary and secondary button colors slightly on hover for a tactile feel. Example: #47B1E8 for Ocean Blue and #FFAA73 for Sunrise Orange.
	•	Gray Overlay: When using background images, apply a #808080 overlay with 50% opacity to ensure that text over the images remains legible.
