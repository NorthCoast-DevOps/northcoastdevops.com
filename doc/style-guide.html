NorthCoast DevOps Style Guide

Color Palette

Light Gray: #f0f0f0

Usage: Primary background color.

Dark Gray: #424245

Usage: Primary text color.

Almost Black: #1d1d1f

Usage: Header text color.

Medium Blue: #2980b9

Usage: Primary button color.

Blue: #3498db

Usage: Button hover color and accents.

Light Gray: #ecf0f1

Usage: Secondary background color (e.g., for feature callouts).

Bright Blue: #00a8ff

Usage: Link color and hover effects.

Typography

The primary font family for all text is:

SF Pro Display, SF Pro Icons, Helvetica Neue, Helvetica, Arial, sans-serif

This font stack provides a modern, clean look across different devices and operating systems.
